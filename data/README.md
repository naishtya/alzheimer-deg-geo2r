# Data

This folder contains references or metadata related to the dataset used in this project.

Dataset information:

- GEO Accession Number: GSE5281
- Organism: Homo sapiens
- Brain region analyzed: Entorhinal Cortex
- Platform: Microarray

Raw data files were not downloaded locally, as the differential expression analysis was performed directly using the GEO2R web-based tool.

The dataset can be accessed from:
https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE5281
